# FluxENgine-MAXSS
This is **version 2** of code to run FluxEngine calculations for sea-air CO2 flux during tropical cyclones. In this version of code we further developed the code to calculate sea-air CO2 fluxes for 5 N.Atlantic test storms on a local laptop computer. The major change in this version of code is that Fluxengien now uses a moving mask that is based on storm life stage at individual grid squares.
