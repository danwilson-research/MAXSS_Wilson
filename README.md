# FluxENgine-MAXSS
This is **version 3** of code to run FluxEngine calculations for sea-air CO2 flux during tropical cyclones. 
In this version of code we are aiming to get ready for running FluxEngine calculations on larger numbers of 
tropical storms and on the High Performance Computer. 

Development goals for this version of code also inlcudes switching to a new gas transfer parameterisation that can
better account for high speed winds.
